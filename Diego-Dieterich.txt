  1. What is Semantic HTML? 
        Semantic HTML is used to apply meaning and context to a webpage aside from appaerance. This makes code easier to read and understand.

    2. What is HTML used for? 
        HTML is used to create a foundation for a webpage, containing the contents and information of that webpage. This includes images, headers, footers, and paragraphs of text.

    3. What is an attribute and where do we put it? 
        An attribute is used to define a property in HTML. An example is the 'href' attribute, which allows to link a property to other webpages.

    4. What is the h1 tag used for? How many times should I use it on a page?
        The h1 tag presents the most important information on a page, providing a large text header. This should only be used once on any given page.

    5. Name two tags that have required attributes
        Both the <a> and <img> tags require attributes

    6. What do we put in the head of our HTML document? 
        Generally a <div> container to easily edit the most general appearances of a page

    7. What is an id? 
        ID tags are used to name a single tag on an HTML page that can later be styled via CSS.               

    8. What elements can I add an id to? 
        ID can be used on any semantic HTML tags that can be singled out and altered on the page: <div> <body> <header> <section>

    9. How many times can I use the same id on a page? 
        Good practice would mean that you would only want to use it once on a single HTML file.

    10. What is a class? 
        Similar to an ID, it can be used to specify a particular HTML tag by name, but can be used on multiple elements that fall under a similar catagory

    11. What elements can I add a class to? 
        All the same elements as ID tags, but can be used on several different ones at a time.

    12. How many times can I use the same class on a page? 
        As many times as necesary, meaning it can be used on sepearte contents in the body of a webpage that will share the same stylistic principles

    13. How do I get my link to open in a new tab?
        You can add 'target=blank' on a <a> tag with an 'href' attribute.

    14. What is the alt attribute in the image tag used for? 
        It is used to define the source of the image

    15. How do I reference an id?
        In order to refrence an ID, one adds "id= [tagName]" to a before closing it. 

    16. What is the difference between a section and a div
        A <div> fits in more broader contexts within the page, while <section> can be used to define portions of the page within those <div> tags.

    17. What is CSS used for? 
        CSS is used to customize the contents of HTML.

    18. How to we select an element? Example - every h2 on the page
        You refer to the tag and element from an HTML file and apply alterations bewteen {} like so: h2 {color: red;}

    19. What is the difference between a class and an id? - Give me an example of when I might use each one
        The main diffrence between a class and ID is functionality. They both share a similar purpose, but you should only "id" one element in HTML, while classes can define multiple elements that share similar properties such as multiple paragraphs on a page, or sections of images.

    20. How do we select classes in CSS?
        Classes will be defined as .[tagName] on a page

    21. How do we select a p element with a single class of “human””?
        You type: .human { }

    22. What is a parent child selector? When would this be useful? 
        A parent child selector can be used to specify which HTML tags should be altered. This will take priority over any other another other changes that were made in a more generalized line of code.

    23. How do you select all links within a div with the class of sidebar?
        You type: .side div { }

    24. What is a pseudo selector?
        A pseudo selector allows to apply certain conditional changes to an HTML tag

    25. What do we use the change the spacing between lines?
        Within the curly brakets of a chosen HTML tag, you can type: {line-height: 20px;}

    26. What do we use to change the spacing between letters?
        You can type: {letter-spacing:2px;}

    27. What do we use to to change everything to CAPITALS? lowercase? Capitalize?
        You can type: {text-transform:Uppercase;}, and otherwise "capitalize" or "lowercase" instead of "uppercase"

    28. How do I add a 1px border around my div that is dotted and black?
        You can type: div {border: 1px dotted black;}

    29. How do I select everything on the page? 
        You can used the "body" selector

    30. How do I write a comment in CSS?
        You display text between /* and */

    31. How do I find out what file I am in, when I am using the command line? 
        You can type 'pwd' into the commands line

    32. Using the command line - how do I see a list of files/folders in my current folder?
        You can type 'ls' into the commands line

    33. How do I remove a file via the command line? Why do I have to be careful with this? 
        You can type 'rm [fileName]' in to the command line. This will absolutely remove the file, and can break your code or your computer if not careful.

    34. Why should I use version control? 
        If one ever decides that they need to return to a previous version of project, they can do so.

    35. How often should I commit to github?
        As frequently as possible
    36. What is the command we would use to push our repo up to github? 
        git push -u origin master

    37. Walk me through Lambda's git flow. 
        You fork the repository that is currently worked on, create a branch formatted as so: firstName-lastName, and push your new content back on the the remote repository using your branch
